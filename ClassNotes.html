Stage 1: Make your Web Page

HTML - Hypertext Markup Language - the standard markup language used to create web pages.
HTML Elements are either inline or block.
Tags are currently refered to as elements
CSS - "Cascading Style Sheets" - style sheet language used for describing the look and formatting of a document written in a markup language.
CSS allows you to create rules that specifies how the content will apper.
Using special text editors like sublime, scratchpadio, and codepen make programming easier.
CSS is used to change font types, font size, background colors, font colors, margins, padding, borders, font-alignment....etc
Structure in HTML - described as "tree-like-structure" - means that elements can have other elements inside of them. When looking at the code you can see this as indentions.
Don't duplicate yourself. The more duplication within code make is harder to maintain and makes it more difficult to realize that if a change is made in one location it may have to be made in another.
